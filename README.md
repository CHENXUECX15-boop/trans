# 医学术语四源查询台

这是一个本地查询网页。为了让词典结果直接显示在页面下方，请先启动本地服务。

## 使用方式

1. 双击 `start-server.bat`。
2. 在浏览器打开 `http://127.0.0.1:8787`。
3. 输入英文或中文医学术语。
4. 点击“查询全部”，网页会在当前页面下方优先显示欧路词典“中文百科”第一段，并复制需要手动查询的词条。
5. 将欧路词典、湘雅医学专业词典、CNKI 翻译助手和《英中医学辞海》的译法粘贴到对应文本框。
6. 点击“生成汇总”，得到推荐译法、来源对照和可复制的引用汇总。

## CSV 批量导入

支持 `.csv` 或 `.tsv`，列名如下：

```csv
term,source,translation,note
apoptosis,eudic,细胞凋亡,词条例句提示
```

`source` 可填：

- `eudic`
- `xiangya`
- `cnki`
- `medical-sea`

为了尊重词典版权，页面不内置受版权保护的词典正文，只保存你自行录入或导入的检索记录。
